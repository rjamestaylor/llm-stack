# IMPORTANT NOTICE: Benchmarking Moved to llm-bench

The benchmarking functionality previously contained in this directory has been moved to a dedicated repository: [llm-bench](https://github.com/llm-bench).

## What This Means

- All scripts in this directory are maintained for legacy purposes only
- New features and updates will only be available in the llm-bench repository
- For the latest benchmarking tools, please use the llm-bench repository

## Available in llm-bench

The llm-bench repository provides enhanced versions of all the functionality previously available here:

- Benchmark scripts for testing model performance
- Visualization tools for analyzing results
- Performance and memory analysis utilities
- Advanced features for GPU metrics and custom benchmarking
- Comprehensive documentation

## Migration

If you've been using these benchmarking tools, we recommend migrating to llm-bench for continued support and new features.

Please refer to the [README.md](README.md) in this directory for more information.
